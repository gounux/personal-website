Made with [`zensical`](https://zensical.org/).

- install [`uv`](https://docs.astral.sh/uv/) :

```sh
python3 -m pip install uv
```

- clone and install dependencies :

```sh
uv sync
```

- serve website locally :

```sh
uv run zensical serve
```
